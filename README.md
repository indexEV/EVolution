# EV'olution

Competitive Pokémon EV spread optimizer built for serious players.

EV'olution helps you generate **efficient EV spreads** by solving real in-game scenarios:
- Survive specific attacks
- Secure OHKOs
- Outspeed key threats
- Optimize bulk vs offense tradeoffs

Powered by Smogon’s official damage calculation engine.

---

## 🚀 Features

- 🔍 Brute-force EV optimization
- ⚔️ Threat-based calculations (real matchups)
- 🧠 Smart constraint system (speed tiers, survival, KOs)
- 🌦️ Field conditions support (weather, terrain, etc.)
- 🎯 Built for Gen 9 Doubles OU
